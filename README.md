# 实训题目

LLVM 编译器的初步学习

# 课题简介

LLVM 是一个著名的开源编译器项目、库、工具，本课题从
LLVM 源代码编译与安装起步，继而从"Hello world!"起步
逐步学习基于 LLVM 的软件编写。

# 目标要求

1. LLVM 源代码编译与安装;
2. 基于 LLVM 的开发入门.

# 初步目标

## 1.LLVM 源代码编译安装

工作环境:Windows

方便个人操作,但使用方面可能较为繁琐
工具:
1. git 2.38.1.windows.1
    - 用于下载llvm源码
2. getgnuwin32 0.6.3
    - 用于在Windows上实现Linux指令
3. Cmake 3.25.0
    - 通过源码编译生成搭建工作环境的项目
4. Microsoft Visual Studio 2022
    - 生成工作环境
5. python 3.10.0
    - 用于自动化测试,搭建过程中未使用
