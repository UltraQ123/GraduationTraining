# 实训题目

LLVM 编译器的初步学习

# 课题简介

LLVM 是一个著名的开源编译器项目、库、工具，本课题从
LLVM 源代码编译与安装起步，继而从"Hello world!"起步
逐步学习基于 LLVM 的软件编写。

# 目标要求

1. LLVM 源代码编译与安装;
2. 基于 LLVM 的开发入门.
